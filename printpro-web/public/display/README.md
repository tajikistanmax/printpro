# Фото для второго экрана (дисплей покупателя)

Положите сюда 3 изображения — они автоматически подставятся в дизайны второго
экрана (Настройки → Касса → «Дизайн второго экрана»). Если файла нет, показывается
брендовая заглушка (градиент + логотип) — ничего не ломается.

| Файл | Дизайн | Что на фото (как в макете) |
|------|--------|-----------------------------|
| `showcase.jpg`     | Витрина товара | Крупный план стопки визиток / товар |
| `catalog-hero.jpg` | Каталог услуг  | Креативная композиция печати (постер, стакан, буклеты) |
| `promo-hero.jpg`   | Промо-витрина  | Рабочий стол типографии (кружка, брошюры, визитки) |

Рекомендации:
- Формат: **JPG** (именно эти имена) или PNG (тогда скажите — поправлю расширение в коде).
- Ориентация — горизонтальная, размер от **1200×800** (3:2) и больше.
- Вес желательно < 1 МБ (быстрее грузится на экране у кассы).

После добавления файлов обновите второй экран (Ctrl+Shift+R).
