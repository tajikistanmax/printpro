# PrintPro — ориентир для ассистента

Платформа для типографии/полиграфии. Моно-репозиторий: `printpro-api` (NestJS + Prisma + PostgreSQL), `printpro-web` (Next.js).

## ВАЖНО: при начале/возобновлении работы
**Сначала прочитай [`docs/СТАТУС-ПРОЕКТА.md`](docs/СТАТУС-ПРОЕКТА.md)** — там актуальное «где мы остановились», что готово и что делать дальше. Обновляй этот файл в конце каждой сессии (дата + последний коммит + что сделано/осталось).

Полное ТЗ: [`docs/00-ТЗ-PrintPro.md`](docs/00-ТЗ-PrintPro.md). Деплой: [`docs/02-deploy-render.md`](docs/02-deploy-render.md).

## Ключевое
- Это **типография**, не продуктовый магазин — «магазинных» функций (весовой товар и т.п.) не предлагать; при неоднозначности сначала уточнять у владельца.
- Запуск: см. раздел «Как запустить локально» в `docs/СТАТУС-ПРОЕКТА.md`. Вход `admin`/`admin123`.
- Windows: перед `prisma generate`/`migrate` останавливать dev-сервер API (держит DLL Prisma).
- Владелец работает с разных машин — перед работой `git pull`.
